What is Grrab

Grrab is a Chrome extension. Hover an image on any page and press the clip shortcut. The image is copied to the clipboard immediately, and when signed in it is also saved to the chosen destination and listed in the side panel.

Images smaller than the chosen clip size are enlarged on-device with a bundled super-resolution model; nothing is uploaded for processing.

Install

Install Grrab from the Chrome Web Store. Click the Grrab icon to open the side panel and sign in with Google, then choose a save destination.

Clipping to the clipboard works without signing in. The side panel, saved history, and settings require signing in.

Daily workflow

Hover a clippable image; Grrab highlights it. Press the shortcut. The image lands on the clipboard right away. When signed in, a card also appears in the side panel.

Clip size and AI upscaling

Selectable clip sizes are Original, 512px, 1024px, 1600px, and 3K. Larger sources are resized down; smaller ones are AI-upscaled. Upscaling runs in an offscreen document on your own machine.

Save destinations

Grrab can save clips to:

  • Your Downloads folder
  • A local folder you choose with the File System Access API
  • Google Drive, into a folder named Grrab_files, using the drive.file scope so Grrab can only touch files it created

Save format can be Original, JPG, JPEG, PNG, or WEBP. You can switch destinations at any time from the side panel.

Side panel

Clips appear in one list, grouped by date dividers (Today, Yesterday, older). Each card shows a source label, plus a title and thumbnail.

Each card has exactly three actions: copy the image back to the clipboard, open the original link, and delete. The toolbar has bulk upload for selected clips, and clear-all.

Change the clip shortcut

The default shortcut is Cmd+C on macOS and Ctrl+C on Windows and Linux. In the side panel, click the shortcut button under “Clip Shortcut”, then press the combination you want.

The combination must include Cmd or Ctrl plus a key. Some combinations reserved by the browser are rejected. Press Escape to cancel recording; the reset button restores the default.

Troubleshooting

“Folder access denied” or “Folder not found”
Choose the save folder again from the side panel.
“Write permission was denied for this folder”
Pick a folder you can write to.
“Drive folder not found”
Set the Drive folder again.
“Google sign-in failed”
Sign in again from the side panel.
The shortcut shows an error
The combination must include Cmd or Ctrl, and some browser-reserved combinations are unavailable.
Nothing happens on a page
Reload the page, then hover the image again.

Privacy

Image processing and AI upscaling run on your device; clip bytes are not sent to a server for enhancement. Saved clips and metadata go to your own Google account or the folder you chose.

Read the full policy at grrab.app/privacy or the Korean version at grrab.app/privacy/kr.
